The 2024-2025 frontier: predict not one token but a long chain of working-out, spend more compute at inference, and a model's answers on hard problems jump. The reasoning-model wave -- OpenAI o1/o3, DeepSeek-R1, Google thinking models, Claude's extended/adaptive thinking -- cited as public fact, not minted. Cross-links the-quorum-engine + the AI domain.
